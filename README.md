### 👩🏻‍💻 CHOLOGBOOK Creator

작은 행동(Patch)을 쌓고, <br>
흐름(Minor)을 인지하며, <br>
변화(Major)를 발견하는 구조를 만듭니다.

기록을 대신해주는 것이 아니라, <br>
스스로 변화를 느낄 수 있는 타이밍을 설계합니다.

---

### Project

#### 초록북(CHOLOGBOOK)

Patch → Minor → Major 구조를 기반으로 <br>
행동이 흐름이 되고, 흐름이 변화로 이어지도록 만드는 시스템
